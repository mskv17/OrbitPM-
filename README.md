OrbitPM 

Project Management Platform

"Manage teams, projects and tasks with real-time collaboration."

> This Project is currently under development.

## Tech Stack

### Frontend 

- React
- Redux Toolkit
- React Router
- Bootstrap
- Axios
- React Beautiful DnD (or dnd-kit)
- Socket.io Client

### Backend

- Node.js
- Express
- MongoDB
- Mongoose
- JWT
- Socket.io

### Dev Tools

- ESLint
- Prettier

## Features (Planned)

- User Authentication
- Create & Manage Workspaces
- Boards
- Lists
- Tasks
- Drag & Drop
- Comments
- Due Dates
- Team Collaboration

### Clone the repository


### Install dependencies

```bash

npm install

cd ../backend
npm install
```
